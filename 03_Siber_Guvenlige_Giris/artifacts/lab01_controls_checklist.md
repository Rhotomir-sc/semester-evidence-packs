# Controls Checklist — Portfolio + GitHub Evidence

## Account Security
- [ ] 2FA enabled for GitHub
- [ ] Unique password + password manager
- [ ] Review and revoke old tokens

## Repo Integrity
- [ ] Branch protection for main (where applicable)
- [ ] Avoid committing secrets (.env, keys)
- [ ] Use .gitignore for local artifacts

## Data Hygiene
- [ ] Sanitize screenshots (hide usernames/IPs/emails)
- [ ] Do not upload copyrighted PDFs; summarize instead
- [ ] Keep private notes in private storage, not public repo

## Evidence Quality (HR)
- [ ] Problem → Action → Evidence → Result clearly stated
- [ ] Standard naming (lab01_*.png, lab01_*.csv)
