# System Overview — Student Portfolio Website + GitHub Evidence Repo

## Scope
- Public portfolio website (optional) linking to GitHub repos
- GitHub repositories containing evidence packs (markdown, screenshots, small scripts)

## Assets (what to protect)
- GitHub account access (credentials, tokens)
- Repository integrity (no malicious commits)
- Personal data (email, documents, screenshots)
- Reputation/brand (impersonation risk)

## Actors
- Owner (me)
- HR/recruiters (read-only)
- Attackers: account takeover, repo vandalism, impersonation

## Trust boundaries
- Local machine ↔ GitHub
- Browser ↔ GitHub web
- Optional website ↔ visitors
