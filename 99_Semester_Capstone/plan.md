# Plan — Mini SOC Investigation Pack

## Scope
- Link and summarize the strongest evidence from 5 courses.
- Provide a short, reproducible “investigation flow” that a reviewer can follow.

## Success criteria
- HR understands the portfolio in <30 seconds.
- A reviewer can verify evidence in <5 minutes (links + screenshots + artifacts).
- No copyrighted PDFs stored; only summaries + outputs.

## Inputs
- OS evidence: users/sudo, services/ports, journald sudo events
- Network evidence: baseline PCAP + DNS filter + TLS handshake
- Cyber evidence: STRIDE-lite threat model + risk register + controls checklist
- Law/GRC evidence: week map + risk register + compliance checklist
- Programming evidence: C tool + CSV output
